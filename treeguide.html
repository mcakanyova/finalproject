Tree Guide

Working with items such as morphemes and phrases sometimes requires segmentation and combination of the items into bigger structures. This can be best observed in trees. Below, you can see some examples as well as some links to online tree builders.

If you want to draw trees through a programme, an ideal one is this: yohasebe.com/rsyntaxtree/ You need to know how to present the phrases in brackets. Each phrase needs to be broken down to the smallest elements - heads:
[CP [DP [D My] [NP [AP younger] [NP brother]]] [IP [I s][VP [V read] [NP [NP books ]]]]]

For more practice, we can have a look at the following trees which represent sentences:

  1. My going to Edinburgh is important.
  2. Peter wants him to call me.
  3. She decided yeasterday to do it tomorrow.
  4. I must speak to him.
