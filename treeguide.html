Tree Guide

Working with items such as morphemes and phrases sometimes requires segmentation and combination of the items into bigger structures. This can be best observed in syntactic trees. There are many ways to approach this. Apart from a pen and paper, and Word document lines, there are many programs avaialbel online that will generate the trees for us. Below, you can see some examples as well as some links to online tree builders.

If you want to draw trees through a programme, an ideal one is this: yohasebe.com/rsyntaxtree/ You need to know how to present the phrases in brackets. Each phrase needs to be broken down to the smallest elements - heads. For example, we can have a sentence: My younger brother reads books.
To segment this sentence, we need to identify the subject DP (my younger brother) and the predicate IP (reads books) and continue with further segmentation. We end up with the follwoing structure:
[CP [DP [D My] [NP [AP younger] [NP brother]]] [IP [I s][VP [V read] [NP [NP books ]]]]]
