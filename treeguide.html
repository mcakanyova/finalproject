Tree Guide

Working with items such as morphemes and phrases sometimes requires segmentation and combination of the items into bigger structures. This can be best observed in trees. Below, you can see some examples as well as some links to online tree builders.

If you want to draw trees through a programme, an ideal one is this: https://yohasebe.com/rsyntaxtree/ You need to know how to present the phrases in brackets. Each phrase needs to be broken down to the smallest elements - heads: [S [DP [D My] [NP [AP younger] [NP brother]]] [VP [V reads] [NP [NP books ]]]]
